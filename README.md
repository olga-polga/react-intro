## Synopsis

Demo React js app - to get started with basics

## Prerequisites:
install `node.js`: https://nodejs.org/en/
install `yarn`: https://yarnpkg.com/lang/en/docs/install/

## Install required packages:
```
yarn install
```

## Run locally (http://localhost:3000/)
```
yarn start
```

## Deploy to PCF:
```
yarn run PCF
```
